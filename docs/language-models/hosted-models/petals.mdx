title: Petals
---

To use Probe with a model from Petals, set the `model` flag to begin with `petals/`:

<CodeGroup>

```bash Terminal
probe --model petals/petals-team/StableBeluga2
```

```python Python
from probe import probe

probe.llm.model = "petals/petals-team/StableBeluga2"
probe.chat()
```

</CodeGroup>

# Pre-Requisites

Ensure you have petals installed:

```bash Terminal
pip install git+https://github.com/bigscience-workshop/petals
```

# Supported Models

We support any model on [Petals:](https://github.com/bigscience-workshop/petals)

<CodeGroup>

```bash Terminal
probe --model petals/petals-team/StableBeluga2
probe --model petals/huggyllama/llama-65b
```

```python Python
probe.llm.model = "petals/petals-team/StableBeluga2"
probe.llm.model = "petals/huggyllama/llama-65b"
```

</CodeGroup>

# Required Environment Variables

No environment variables are required to use these models.
