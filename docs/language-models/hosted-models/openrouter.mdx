To use Probe with a model from OpenRouter, set the `model` flag to begin with `openrouter/`:

<CodeGroup>

```bash Terminal
probe --model openrouter/openai/gpt-3.5-turbo
```

```python Python
from probe import probe

probe.llm.model = "openrouter/openai/gpt-3.5-turbo"
probe.chat()
```

</CodeGroup>

# Supported Models

We support any model on [OpenRouter's models page:](https://openrouter.ai/models)

<CodeGroup>

```bash Terminal
probe --model openrouter/openai/gpt-3.5-turbo
probe --model openrouter/openai/gpt-3.5-turbo-16k
probe --model openrouter/openai/gpt-4
probe --model openrouter/openai/gpt-4-32k
probe --model openrouter/anthropic/claude-2
probe --model openrouter/anthropic/claude-instant-v1
```

# Required Environment Variables

Set the following LiteLLM environment variables [(click here to learn how)](https://chat.openai.com/share/1062cdd8-62a1-4aa8-8ec9-eca45645971a) to use these models.

| Environment Variable  | Description  | Where to Find  |
| --------------------- | ------------ | -------------- |
| `OPENROUTER_API_KEY`       | The API key for authenticating to OpenRouter's services. | [OpenRouter Account Page](https://openrouter.ai/keys) |
| `OR_SITE_URL`      | An optional app URL for tracking usage, such as `https://github.com/probe/probe/`. | Your choice |
| `OR_APP_NAME`   | An optional app name for tracking usage, such as `"Probe"`. | Your choice |
