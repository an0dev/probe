title: vLLM
---

To use Probe with vLLM, you will need to:

1. `pip install vllm`
2. Set the api_base flag:

<CodeGroup>

```bash Terminal
probe --api_base <https://your-hosted-vllm-server>
```

```python Python
from probe import probe

probe.llm.api_base = "<https://your-hosted-vllm-server>"
probe.chat()
```

</CodeGroup>

3. Set the `model` flag:

<CodeGroup>

```bash Terminal
probe --model vllm/<vllm-model>
```

```python Python
from probe import probe

probe.llm.model = "vllm/<vllm-model>"
probe.chat()
```

</CodeGroup>

# Supported Models

All models from VLLM should be supported
