title: Perplexity
---

To use Probe with the Perplexity API, set the `model` flag:

<CodeGroup>

```bash Terminal
probe --model perplexity/<perplexity-model>
```

```python Python
from probe import probe

probe.llm.model = "perplexity/<perplexity-model>"
probe.chat()
```

</CodeGroup>

# Supported Models

We support the following completion models from the Perplexity API:

- pplx-7b-chat
- pplx-70b-chat
- pplx-7b-online
- pplx-70b-online
- codellama-34b-instruct
- llama-2-13b-chat
- llama-2-70b-chat
- mistral-7b-instruct
- openhermes-2-mistral-7b
- openhermes-2.5-mistral-7b
- pplx-7b-chat-alpha
- pplx-70b-chat-alpha

<CodeGroup>

```bash Terminal
probe --model perplexity/pplx-7b-chat
probe --model perplexity/pplx-70b-chat
probe --model perplexity/pplx-7b-online
probe --model perplexity/pplx-70b-online
probe --model perplexity/codellama-34b-instruct
probe --model perplexity/llama-2-13b-chat
probe --model perplexity/llama-2-70b-chat
probe --model perplexity/mistral-7b-instruct
probe --model perplexity/openhermes-2-mistral-7b
probe --model perplexity/openhermes-2.5-mistral-7b
probe --model perplexity/pplx-7b-chat-alpha
probe --model perplexity/pplx-70b-chat-alpha
```

```python Python
probe.llm.model = "perplexity/pplx-7b-chat"
```

</CodeGroup>

# Required Environment Variables

Set the following environment variables [(click here to learn how)](https://chat.openai.com/share/1062cdd8-62a1-4aa8-8ec9-eca45645971a) to use these models.

| Environment Variable    | Description                          | Where to Find                                                     |
| ----------------------- | ------------------------------------ | ----------------------------------------------------------------- |
| `PERPLEXITYAI_API_KEY` | The Perplexity API key from pplx-api | [Perplexity API Settings](https://www.perplexity.ai/settings/api) |
